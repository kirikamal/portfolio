Local RAG Document Assistant

A powerful, privacy-focused document question-answering system that runs entirely on your local machine. Upload documents and ask questions about their content using Retrieval-Augmented Generation (RAG) powered by local AI models.

Features

  • Multiple Format Support: PDF, Microsoft Word (.docx), and plain text (.txt) files
  • Intelligent Text Extraction: Specialized processors for each file type
  • Retrieval-Augmented Generation (RAG): Combines vector search with LLM generation
  • Streaming Responses: Real-time answer generation with progressive display
  • Source Attribution: Shows which document chunks were used to generate each answer
  • Context Preview: Hover over sources to view the relevant text passages

Tech Stack
