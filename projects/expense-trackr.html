Expense Trackr

A full-stack expense tracking application built with a FastAPI backend and a Streamlit frontend, designed to demonstrate clean API design, modern Python practices, and rapid UI prototyping.

The backend exposes RESTful endpoints for managing expense data, handling business logic, and integrating with a relational database, while the frontend provides an interactive interface for recording, viewing, and analyzing expenses in real time. The application emphasizes a clear separation of concerns, reproducible local setup, and an architecture that can be easily extended to a React-based frontend in future iterations.

Features

  • Expense categories
  • Expenses breakdown by categories
  • Analytics of monthly expenses

Tech Stack

Python FastAPI MySQL
